Otzar Hachochma Filter

Turns a Windows 11 Pro PC into a locked-down Otzar Hachochma kiosk.

version …

What it does

Creates a passwordless standard account that can run only Otzar Hachochma (plus LibreOffice and a read-only PDF browser), blocks everything else, replaces the desktop with a custom launcher, and keeps AnyDesk remote support working. Your admin account is untouched.

Locks down

  • All other apps & browsing
  • USB storage & phones (MTP)
  • Bluetooth, Task Manager, Run
  • Store / media / games apps

Keeps working

  • Otzar Hachochma (auto-launch)
  • LibreOffice + PDF viewing
  • Printing (print-only)
  • AnyDesk incoming support

Install

On the administrator account, in an elevated PowerShell:

cd "$env:USERPROFILE\Downloads\Otzar Hachochma Filter"
Set-ExecutionPolicy -Scope Process Bypass -Force
.\create.ps1     # make the account, then log in once with 1234 and sign out
.\setup.ps1      # apply the lockdown + launcher

Both scripts self-update from version.xml on startup. Full source on GitHub.
